# Sales Dashboard Analysis (Excel)

## About This Project
This project is an interactive Sales Dashboard built in Microsoft Excel to analyze sales performance across products, regions, and sales representatives.

The dashboard helps users monitor key business metrics and gain insights into sales trends through interactive visualizations and filters.

## What I Did in This Project
- Cleaned and prepared the sales dataset
- Created Pivot Tables for data analysis
- Built KPI cards to display important business metrics
- Designed an interactive dashboard layout
- Developed visualizations for:
  - Product-wise Units Sold
  - Product-wise Total Sales
  - Region-wise Sales Distribution
  - Salesperson Performance Analysis
- Added slicers for dynamic filtering by:
  - Product
  - Sales Person
  - Region

## Key Metrics Tracked
- Total Sales
- Average Sales
- Units Sold
- Total Profit

## Insights Observed
- Smartphone generated the highest sales revenue.
- Novel recorded the highest number of units sold.
- East region contributed the highest sales among all regions.
- Sales performance varied across different sales representatives.
- Product-wise analysis helped identify top-performing products.
